# 通用方法收集

## 一、数组

### 1. 对象数组去重

```javascript
// arr: 需去重的数组；
// key: 对象中唯一标志符
const filterObjArr = (arr, key) => {
  var hash = {};
  arr = arr.reduce(function(item, next) {
    hash[next[key]] ? "" : (hash[next[key]] = true && item.push(next));
    return item;
  }, []);
  return arr;
};

// filterObjArr([{id: 2, name: 'heihei'}, {id: 6, name: 'haha'}, 'id']);
```
